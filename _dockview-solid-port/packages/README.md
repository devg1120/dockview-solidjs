## dockview-core

-   Published as [@arminmajerie/dockview-core](https://www.npmjs.com/package/@arminmajerie/dockview-core) on npm.
-   Contains the core logic for the layout manager.
-   Written entirely in vanilla JavaScript/TypeScript.
-   Mobile-friendly fork with touch support.

## dockview-solid

-   Published as [@arminmajerie/dockview-solid](https://www.npmjs.com/package/@arminmajerie/dockview-solid) on npm.
-   Depends on `@arminmajerie/dockview-core`.
-   Exports a pure SolidJS wrapper (no React dependency).
